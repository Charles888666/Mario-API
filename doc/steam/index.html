Steam游戏库

获取你的Steam游戏仓库数据，适用于二次开发，需将你的库设置为公开，详见：Steam-page

请求URL：https://your.domain/steam

请求方式：GET

返回格式：JSON

请求示例：https://your.domain/steam?id=76561198849944519&type=all

参数：

参数名 类型 示例值 描述
id 必选 SteamID 不知道如何获取的可以参考 https://m1314.cn/327.html
type 可选 all/recent all：全部游戏；recent：最近游玩；默认为all
